# 每周一数据更新标准操作手册（6步照做版）

> 不讲原理，只讲执行。每周一就按下面 6 步做。

## 第 1 步：进入项目目录

```bash
cd "/Users/weiwei/餐厅周度经营看板_改"
```

## 第 2 步：生成数据（默认自动从上次结果往后算）

```bash
PYTHONPATH=. python3 scripts/run_monday_update.py
```

说明：脚本会自动取当前 `ui_payload` 最后一周的下一周作为边界，只追加新周，不重算已生成历史。

如需手动指定边界（例如修复某段历史），用：

```bash
PYTHONPATH=. python3 scripts/run_monday_update.py --recompute-from 2026-03-16
```

## 第 3 步：快速看关键结果（可选）

```bash
python3 - << 'PY'
import json
o=json.load(open('data/warehouse/ui_payload.json',encoding='utf-8'))
for s in ['夕佳悦','紫竹']:
    print('\n',s)
    for wk in ['2026-03-16','2026-03-23','2026-03-30','2026-04-06','2026-04-13']:
        r=o['stores'][s]['weeks'][wk]['productDetails']['returns']
        print(wk,'退/换菜',r['count'])
PY
```

## 只有一个异常处理（push 失败）

如果推送失败，先换网络（手机热点），再执行：

```bash
git -c http.version=HTTP/1.1 push origin main
```

## 注意（非常重要）

- 不要执行带 `#` 的注释行。
- 如果你这周要从别的周开始重算（不是 `2026-04-06`），只改第 2 步命令里的日期。
